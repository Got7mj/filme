# Portal de Filmes — Mini Projeto

Este repositório contém uma evolução didática do projeto "Portal de Filmes":

- `static/` — HTML/CSS/JS inicial
- `react/` — exemplo simples `App.jsx`
- `app/` — esqueleto Next.js (app router) com API interna

Para rodar o Next.js:

```bash
cd portal-filmes
npm install
npm run dev
```

Abra `http://localhost:3000` após iniciar.
